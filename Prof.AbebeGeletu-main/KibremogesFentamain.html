// iMPC Pipeline

The Intelligent MPC Framework

A unified embedded pipeline bridging deep learning perception, adaptive MPC theory, and real-time control — all onboard NVIDIA Jetson.

01

Monocular Camera Perception

RGB images captured at 30 Hz. Dynamic perception window applied with heading-aligned crop. Resize 224×224, normalise, FP16 tensor preparation.

ResNet18TensorRT FP1611ms
02

Two-Point Keypoint Regression

ResNet18 regresses trajectory waypoints (x₁,y₁), (x₂,y₂) per frame. TensorRT FP16 inference at 11ms. Perception confidence cᵢ ∈ [0,1] output.

CNN RegressionMAE ≤ 4.1px
03

Trajectory Planning & Curvature

Pixel coordinates → world frame. Waypoint generation with curvature κ estimation. Adaptive horizon: f = fn(κₘₐₓ) exponential law.

Kinematic Bicycle2ms
04

Adaptive Horizon MPC Solve

Precomputed gain matrix K = (MᵀWM + R)⁻¹MᵀW offline. O(n³) → O(n²) per step. 6–20× speedup. Perception-confidence-weighted cost.

iMPC15ms solve30Hz
05

Actuator Commands

Optimal steering angle δ* and throttle a* dispatched to vehicle. σ_δ ≤ 3.5° actuation smoothness. Total pipeline Tₜₒₜₐₗ ≈ 28.5ms.

δ* steeringa* throttleRMSE 0.033m

Platform Architecture

// Technology Stack

Built On Solid Foundations

Embedded Perception

ResNet18 with TensorRT FP16 optimisation. Two-point trajectory keypoint regression trained on custom track dataset. Dynamic perception window aligned to vehicle heading.

PyTorchTensorRTResNet18FP16

Adaptive MPC Core

Perception-aware multi-layer MPC with exponential horizon adaptation. Precomputed gain matrix reduces online solve from O(n³) to O(n²), enabling 30 Hz control on Jetson.

iMPCCVXPYNumPyC++ solver

Vehicle Model

Kinematic bicycle model with slip angle β = arctan(lr/(lf+lr)·tan(δ)). State: [x, y, ψ, v]. Valid for speeds up to 3 m/s on the current platform.

Bicycle ModelODE45Linearisation

Server & API

WebSocket-based control gateway with TLS encryption, token authentication, and per-vehicle parameter isolation. MQTT telemetry broker for low-latency state streaming.

WebSocketMQTTFastAPIPython

Hardware Platform

NVIDIA Jetson Nano/Xavier SoC. JetRacer chassis. No GPS, no external cloud dependency — fully standalone 28.5ms pipeline running onboard.

JetsonJetRacerROS2PWM/CAN

Web Dashboard

Real-time telemetry visualisation with Chart.js. Live simulation of iMPC vs PID vs Fixed-MPC. Remote D-pad control, parameter sliders, and connection manager.

Chart.jsCanvas APIWebSocketVanilla JS
